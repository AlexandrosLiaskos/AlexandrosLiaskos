# Alexandros Liaskos

**Geoscientist & Software Engineer** building tools at the intersection of earth sciences, web development, and systems programming.

I create practical software for geospatial analysis, coastal research, and developer workflows — from satellite shoreline extraction to formal verification in Lean 4.

<br>

## Geoscience & Remote Sensing

| Project | Description |
|---------|-------------|
